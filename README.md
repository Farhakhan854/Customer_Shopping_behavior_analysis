# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 customer purchases. The objective is to identify purchasing patterns, customer segments, product preferences, and revenue-driving factors that can help businesses make data-driven decisions.

The project combines **Python**, **SQL**, and **Power BI** to perform data cleaning, exploratory data analysis (EDA), business analysis, and interactive dashboard visualization.

---

## 🎯 Objectives

- Analyze customer purchasing behavior
- Identify high-value customer segments
- Understand the impact of discounts and subscriptions
- Discover top-performing products and categories
- Generate actionable business recommendations

## 🧹 Data Cleaning & Preparation

### Data Exploration
- Used `df.info()` and `df.describe()` to understand dataset structure and statistics.

### Missing Value Treatment
- Filled missing values in the Review Rating column using category-wise median ratings.

### Column Standardization
- Renamed columns using snake_case naming conventions.

### Feature Engineering
Created:
- `age_group`
- `purchase_frequency_days`

### Data Consistency Check
- Verified redundancy between `discount_applied` and `promo_code_used`
- Removed unnecessary columns

## 📈 SQL Business Analysis

### 1. Revenue by Gender
Compared revenue generated by male and female customers.

### 2. High-Spending Discount Users
Identified customers who used discounts while maintaining above-average spending.

### 3. Top Rated Products
Determined products with the highest average customer ratings.

### 4. Shipping Type Analysis
Compared average purchase amounts between Standard and Express shipping methods.

### 5. Subscriber vs Non-Subscriber Analysis
Evaluated spending behavior and revenue contribution based on subscription status.

### 6. Discount Dependency Analysis
Identified products most frequently purchased with discounts.

### 7. Customer Segmentation
Segmented customers into:
